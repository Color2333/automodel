# Meshy AutoModel

Blender add-on for Meshy GLB / USDZ model review, classification, repair workflow tracking, and export management.

This repository stores Meshy Blender workflow plugins. The default `main` branch contains the AutoModel plugin (`Meshy-AutoModel`, version 4.6.7).

## Branches

- `main`: AutoModel workflow plugin for general GLB / USDZ model handling.
- `logoslot`: Logo / Slot focused GLB review and repair workflow plugin.

`LogoSlot_Boolean_Fixer` is not included yet.

## What It Does

- Loads model batches from a selected source directory.
- Tracks review status and operator progress.
- Supports categorized exports and progress reports.
- Provides utility actions for cleanup, review, and model workflow management inside Blender.

## Install

Use the source files from the branch you need, or package the branch contents into a zip and install it as a Blender add-on.

Zip archives and Python cache files are intentionally excluded from this repository.
